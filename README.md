# My-think

## 🤖 Robot Project Ideas

파이썬 기초 문법과 조건문·반복문 학습을 바탕으로,
**로봇 제어와 연결할 수 있는 프로젝트 아이디어**를 정리했습니다.

---

### 1️⃣ 장애물 회피 로봇 (Obstacle Avoidance Robot)

#### 📌 프로젝트 개요

초음파 센서를 이용해 전방의 장애물과의 거리를 측정하고,
장애물이 가까우면 **정지하거나 방향을 전환**하는 자율 주행 로봇입니다.

#### 🔧 구현 기능

* 초음파 센서를 이용한 거리 측정
* 일정 거리 이하일 경우 정지
* 좌·우 회전을 통한 장애물 회피

#### 🛠 사용 기술

* Arduino / Raspberry Pi
* 초음파 센서 (HC-SR04)
* DC 모터, 모터 드라이버
* Python 또는 Arduino C

#### 📘 학습 포인트

* 조건문을 활용한 상황 판단
* 센서 값에 따른 로봇 행동 제어

---

### 2️⃣ 로봇 행동 결정 시스템 (Robot Action Decider)

#### 📌 프로젝트 개요

거리와 배터리 상태를 종합적으로 판단하여
로봇의 행동을 결정하는 **상태 기반 제어 로직** 프로젝트입니다.

#### 🔧 구현 기능

* 장애물 거리 입력
* 배터리 잔량 확인
* 상황에 따른 행동 출력 (전진 / 정지 / 충전 필요)

#### 🛠 사용 기술

* Python
* 조건문 (`if / elif / else`)

#### 📘 학습 포인트

* 다중 조건 판단 로직 설계
* 실제 로봇 제어로 확장 가능한 구조 이해

---

### 3️⃣ 라인 트레이싱 로봇 (Line Tracer Robot)

#### 📌 프로젝트 개요

바닥의 검은 선을 인식하여 선을 따라 이동하는 로봇입니다.

#### 🔧 구현 기능

* IR 센서를 이용한 바닥 색상 인식
* 선 위치에 따른 좌·우 모터 제어

#### 🛠 사용 기술

* IR 센서
* Arduino
* DC 모터

#### 📘 학습 포인트

* 반복문을 활용한 지속적인 센서 확인
* 조건 분기에 따른 모터 제어 로직

---

### 4️⃣ 거리 기반 자동 정지 로봇

#### 📌 프로젝트 개요

로봇이 일정 거리 이상 접근하면 자동으로 멈추는 **안전 로봇** 프로젝트입니다.

#### 🔧 구현 기능

* 거리 센서 입력
* 안전 거리 이하 접근 시 자동 정지

#### 🛠 사용 기술

* Python
* 초음파 센서

#### 📘 학습 포인트

* 센서 데이터 처리
* 안전 로직 설계

---

### 5️⃣ 센서 데이터 기록 로봇

#### 📌 프로젝트 개요

센서로 측정한 거리 또는 온도 값을
파일로 저장하여 분석하는 로봇 시스템입니다.

#### 🔧 구현 기능

* 센서 데이터 수집
* 시간별 데이터 저장

#### 🛠 사용 기술

* Python
* 파일 입출력

#### 📘 학습 포인트

* 데이터 관리 기초
* 로봇 + 소프트웨어 융합 사고

---

# My-think

## 🤖 Robot Project Idea

파이썬 기초 문법과 조건문·반복문 학습을 바탕으로,
**실생활의 불편함을 해결하는 로봇 제어 프로젝트 아이디어**를 정리했습니다.

이 프로젝트는 단순한 아이디어 제시가 아니라,
향후 실제 하드웨어 제작과 센서 연동까지 확장 가능한 포트폴리오를 목표로 합니다.

## ♻️ 분리수거 로봇 (Recycling Robot)

### 📌 프로젝트 개요

집에서 분리수거를 할 때마다 부모님, 특히 **엄마가 분리수거 신부름을 자주 시키는 상황**이 반복되었습니다.
이 과정이 번거롭고 귀찮다고 느껴졌고,
이 불편함을 기술로 해결해 보고 싶다는 생각에서 **분리수거 로봇** 아이디어를 떠올리게 되었습니다.

이 로봇은 쓰레기의 종류를 인식하고,
종류에 맞는 위치로 자동 분류 및 이동을 수행하는 것을 목표로 합니다.

---

### 🔧 구현 기능

* 카메라 또는 센서를 이용한 쓰레기 감지
* 쓰레기 종류 분류 (일반 / 플라스틱 / 캔 / 종이 등)
* 분류 결과에 따른 이동 또는 투입 동작
* 지정 위치 도착 시 자동 정지

---

### 🛠 사용 기술

* Raspberry Pi 또는 Arduino
* 카메라 모듈 또는 색상 / 무게 센서
* DC 모터, 서보 모터
* Python

---

### 📘 학습 포인트

* 조건문을 활용한 쓰레기 종류 판별 로직
* 반복문을 이용한 지속적인 센서 데이터 확인
* 현실 문제를 해결하는 알고리즘 사고력
* 하드웨어 제어와 소프트웨어 로직의 연결 이해

# ♿ 계단 / 턱 인식 보조 로봇 (Stair & Step Safety Assistant)

보행 약자를 위한 **계단·턱 사전 인식 보조 로봇** 설계 포트폴리오이다.
센서를 통해 바닥 높이 변화를 감지하고, 진동·소리로 위험을 즉시 알린다.

---

## 1. 설계 계기 (기획 동기)

* 보행 약자는 작은 턱이나 계단에서도 **낙상 사고 위험**이 큼
* 시각장애인은 **직접 접촉 전까지** 바닥 상태 인지가 어려움
* 사고 직전 상황에서는 **대응 시간 부족**

➡️ 시야에 의존하지 않고, **센서로 위험을 미리 감지**하는 장치 필요

---

## 2. 문제 정의

기존 보행 보조 도구의 한계

* 지팡이는 **직접 닿아야** 위험 인지 가능
* 계단·턱을 **너무 가까이 가서야** 감지
* 낙상 직전 상황에서 **경고가 늦음**

해결 목표

* 위험 요소 **사전 감지**
* 시각 없이도 **촉각·청각 경고 제공**

---

## 3. 작품 개요

### 작품명

* 계단 / 턱 인식 보조 로봇

### 작품 목적

* 바닥 높이 변화(턱, 계단, 낙하 지점) 실시간 감지
* 위험 거리 계산 후 진동·소리로 경고
* 보행 중 낙상 사고 예방

### 적용 대상

* 시각장애인
* 노약자
* 휠체어 사용자

---

## 4. 설계 아이디어 및 구조

본 작품은 **ToF 거리 센서**를 이용해 전방 바닥과의 거리를 지속적으로 측정한다.

### 거리 변화 패턴

* 평지 → 거리값 일정
* 아래 계단 / 낙하 → 거리값 **급격히 증가**
* 턱 / 연석 → 거리값 **급격히 감소**

➡️ 거리 변화량(Δ거리)을 분석하여 위험 여부 판단

센서는 바닥을 향해 **약 40도 각도**로 고정하여
바닥 높이 변화를 효율적으로 감지하도록 설계하였다.

---

## 5. 사용 기술 및 부품

### 하드웨어

* ESP32 / Arduino : 센서 및 출력 제어
* ToF 거리 센서 (VL53L0X)
* 진동 모터 : 촉각 경고
* 부저(Buzzer) : 청각 경고
* 배터리 전원 시스템

### 소프트웨어

* 거리 데이터 평균화 (노이즈 감소)
* 거리 변화량 계산 알고리즘
* 위험 단계별 출력 제어 로직
* 조건문·반복문 기반 실시간 처리

---

## 6. 동작 방식

1. 센서가 전방 바닥과의 거리 측정
2. 이전 값과 현재 값 비교 → 변화량 계산
3. 설정된 임계값 초과 시 위험 판단
4. 위험 수준에 따라

   * 진동 패턴 출력
   * 경고음 패턴 출력
5. 사용자는 시각 정보 없이 위험 인지

---

## 7. 작품의 장점 및 차별성

* 시각에 의존하지 않는 **촉각·청각 중심 설계**
* 복잡한 AI 없이 **센서 데이터만으로 구현 가능**
* 소형·저전력 구조로 **휴대성 우수**
* 지팡이·휠체어 등 다양한 보조기구에 부착 가능

---

## 8. 테스트 및 검증 방법

* 학교 계단, 인도 연석 등 실제 환경 실험
* 감지 성공률 및 반응 거리 측정
* 평지 오작동 비율 분석
* 임계값 조정으로 정확도 개선

---

## 9. 한계점

* 표면 반사율에 따른 거리 오차 가능성
* 단일 센서 사용 시 좌우 방향 정보 부족
* 복잡한 지형에서 판단 정확도 저하

---

## 10. 향후 확장 계획

### 1) 센서 확장

* 좌·우 2개 센서 → 방향 인식
* IMU 추가 → 사용자 움직임 분석

### 2) 스마트 기능

* Bluetooth 연동 → 스마트폰 알림
* 위험 기록 저장 → 보호자 확인

### 3) 지능화

* 머신러닝 기반 위험 패턴 학습
* 환경별 자동 임계값 조정

---

## 11. 기대 효과

본 프로젝트는 기술을 통해 **사회적 약자의 안전을 지원**하는 것을 목표로 한다.
고령화 사회에서 요구되는 스마트 보행 보조 기술의 **기초 모델**로 활용 가능하다.
