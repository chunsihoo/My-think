# My-think

## 🤖 Robot Project Ideas

파이썬 기초 문법과 조건문·반복문 학습을 바탕으로,
**로봇 제어와 연결할 수 있는 프로젝트 아이디어**를 정리했습니다.

---

### 1️⃣ 장애물 회피 로봇 (Obstacle Avoidance Robot)

#### 📌 프로젝트 개요

초음파 센서를 이용해 전방의 장애물과의 거리를 측정하고,
장애물이 가까우면 **정지하거나 방향을 전환**하는 자율 주행 로봇입니다.

#### 🔧 구현 기능

* 초음파 센서를 이용한 거리 측정
* 일정 거리 이하일 경우 정지
* 좌·우 회전을 통한 장애물 회피

#### 🛠 사용 기술

* Arduino / Raspberry Pi
* 초음파 센서 (HC-SR04)
* DC 모터, 모터 드라이버
* Python 또는 Arduino C

#### 📘 학습 포인트

* 조건문을 활용한 상황 판단
* 센서 값에 따른 로봇 행동 제어

---

### 2️⃣ 로봇 행동 결정 시스템 (Robot Action Decider)

#### 📌 프로젝트 개요

거리와 배터리 상태를 종합적으로 판단하여
로봇의 행동을 결정하는 **상태 기반 제어 로직** 프로젝트입니다.

#### 🔧 구현 기능

* 장애물 거리 입력
* 배터리 잔량 확인
* 상황에 따른 행동 출력 (전진 / 정지 / 충전 필요)

#### 🛠 사용 기술

* Python
* 조건문 (`if / elif / else`)

#### 📘 학습 포인트

* 다중 조건 판단 로직 설계
* 실제 로봇 제어로 확장 가능한 구조 이해

---

### 3️⃣ 라인 트레이싱 로봇 (Line Tracer Robot)

#### 📌 프로젝트 개요

바닥의 검은 선을 인식하여 선을 따라 이동하는 로봇입니다.

#### 🔧 구현 기능

* IR 센서를 이용한 바닥 색상 인식
* 선 위치에 따른 좌·우 모터 제어

#### 🛠 사용 기술

* IR 센서
* Arduino
* DC 모터

#### 📘 학습 포인트

* 반복문을 활용한 지속적인 센서 확인
* 조건 분기에 따른 모터 제어 로직

---

### 4️⃣ 거리 기반 자동 정지 로봇

#### 📌 프로젝트 개요

로봇이 일정 거리 이상 접근하면 자동으로 멈추는 **안전 로봇** 프로젝트입니다.

#### 🔧 구현 기능

* 거리 센서 입력
* 안전 거리 이하 접근 시 자동 정지

#### 🛠 사용 기술

* Python
* 초음파 센서

#### 📘 학습 포인트

* 센서 데이터 처리
* 안전 로직 설계

---

### 5️⃣ 센서 데이터 기록 로봇

#### 📌 프로젝트 개요

센서로 측정한 거리 또는 온도 값을
파일로 저장하여 분석하는 로봇 시스템입니다.

#### 🔧 구현 기능

* 센서 데이터 수집
* 시간별 데이터 저장

#### 🛠 사용 기술

* Python
* 파일 입출력

#### 📘 학습 포인트

* 데이터 관리 기초
* 로봇 + 소프트웨어 융합 사고

---

# My-think

## 🤖 Robot Project Idea

파이썬 기초 문법과 조건문·반복문 학습을 바탕으로,
**실생활의 불편함을 해결하는 로봇 제어 프로젝트 아이디어**를 정리했습니다.

이 프로젝트는 단순한 아이디어 제시가 아니라,
향후 실제 하드웨어 제작과 센서 연동까지 확장 가능한 포트폴리오를 목표로 합니다.

---

## ♻️ 분리수거 로봇 (Recycling Robot)

### 📌 프로젝트 개요

집에서 분리수거를 할 때마다 부모님, 특히 **엄마가 분리수거 신부름을 자주 시키는 상황**이 반복되었습니다.
이 과정이 번거롭고 귀찮다고 느껴졌고,
이 불편함을 기술로 해결해 보고 싶다는 생각에서 **분리수거 로봇** 아이디어를 떠올리게 되었습니다.

이 로봇은 쓰레기의 종류를 인식하고,
종류에 맞는 위치로 자동 분류 및 이동을 수행하는 것을 목표로 합니다.

---

### 🔧 구현 기능

* 카메라 또는 센서를 이용한 쓰레기 감지
* 쓰레기 종류 분류 (일반 / 플라스틱 / 캔 / 종이 등)
* 분류 결과에 따른 이동 또는 투입 동작
* 지정 위치 도착 시 자동 정지

---

### 🛠 사용 기술

* Raspberry Pi 또는 Arduino
* 카메라 모듈 또는 색상 / 무게 센서
* DC 모터, 서보 모터
* Python

---

### 📘 학습 포인트

* 조건문을 활용한 쓰레기 종류 판별 로직
* 반복문을 이용한 지속적인 센서 데이터 확인
* 현실 문제를 해결하는 알고리즘 사고력
* 하드웨어 제어와 소프트웨어 로직의 연결 이해

---

## ✨ 마무리

이 프로젝트는
**"귀찮고 불편했던 일상 속 경험" → "문제 인식" → "로봇으로 해결"**이라는 흐름으로 기획되었습니다.

파이썬 기초 문법 학습을 시작으로,
조건문·반복문을 실제 로봇 제어 상황에 적용해 보고,
추후에는 실제 분리수거 로봇 제작까지 확장하는 것을 목표로 합니다.

이 저장소는 저의 문제 해결 과정과 성장 기록을 담은 포트폴리오입니다.
